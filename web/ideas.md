## 아이디어 A: 경남 남해안 블루 - 해양 산업 도시 테마

**Design Movement**: 산업적 미니멀리즘 + 해양 도시 정체성
**Core Principles**:
- 경남의 산업 수도 이미지 (창원, 거제, 통영)를 반영한 강건한 레이아웃
- 남해안의 깊고 진한 청색 계열로 신뢰와 전문성 표현
- 노션 스타일의 정보 밀도 높은 구조 유지

**Color Philosophy**: 경남 남해의 짙은 청색(#0A2342)을 기반으로, 항공우주산업의 미래지향적 청록색(#00B4D8) 포인트 사용. 배경은 어두운 네이비 계열로 고급스러움 강조.

**Layout Paradigm**: 좌측 고정 사이드바 + 우측 메인 콘텐츠 영역. 우상호 페이지와 동일한 노션 스타일 레이아웃.

**Signature Elements**:
- 경남 지도 실루엣 배경 그래픽
- 항공우주/방산 산업을 상징하는 기하학적 패턴
- 남해안 파도 모티프의 구분선

**Interaction Philosophy**: 호버 시 카드가 살짝 부상하는 효과, 데이터 로딩 시 부드러운 페이드인
**Animation**: 페이지 전환 시 슬라이드 효과, 숫자 카운트업 애니메이션
**Typography System**: Noto Sans KR Bold (제목) + Noto Sans KR Regular (본문)
